# Agent Memory

이 파일은 에이전트가 사용자에 대해 학습한 내용을 기록하는 장기 메모리다.
(선호, 반복되는 피드백, 역할 정의, 도구 사용에 필요한 정보 등)

## User
- 월드컵 경기 결과를 특정 날짜 기준으로 네이버에서 찾고, 화면 캡처 이미지 파일로 받고 싶어 한다.

## Preferences
- 입력이 "7일"처럼 날짜만 들어오면 현재 년월 기준으로 해석한다.
- 입력이 "6월 30일"처럼 월과 일이 들어오면 현재 년 기준으로 해석한다.
- Browser MCP로 네이버 페이지를 열고, Filesystem MCP로 스크린샷 이미지를 저장한다.
- 생성된 이미지 파일은 1MB를 넘지 않도록 한다.
- 저장 위치는 가능하면 workspace/screenshots/ 아래에 둔다.
- 네이버 2026 북중미 월드컵 스코어 확인할 수 있는 url은 https://m.sports.naver.com/fifaworldcup2026/schedule?date=2026-07-07 이다. URL 맨 뒤에 있는 date parameter를 변경해서 원하는 날짜의 스코어를 캡쳐할 수 있다.

## Notes
- 사용자가 날짜만 주면 바로 조회를 시도하고, 필요하면 최소한의 확인만 한다.
- 결과물이 있으면 파일 경로와 한 줄 요약을 함께 전달한다.
